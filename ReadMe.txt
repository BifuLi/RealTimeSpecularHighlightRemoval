This is a C++/OpenCV/CUDA application for real-time specular highlight removal in still images. 

To enable real-time specular highlight removal using GPU, uncomment the line #define REMOVE_SPECULAR_HIGHLIGHT_USING_CUDA in include/SpecularHighlightRemoval/useCUDA.h.

The provided source codes are in public domain and can be downloaded for free. In return, whenever you use it on your own application or research, please let me know or cite my work "Real-Time Specular Highlight Removal using Efficient Pixel Clustering", accepted for publication in the proceedings of the Conference on Graphics, Patterns and Images (SIBGRAPI 2018).
